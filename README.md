# Magento 2 Bookstore Module

A custom Magento 2 Bookstore module developed to manage books and authors with
frontend listing and full admin CRUD functionality.

# Features
- Book listing on frontend
- Admin panel book management (CRUD)
- Author management & relations
- UI Components grid & forms
- Widget support for book listing
- Live search using JavaScript
- LESS-based styling
- Magento 2 coding standards followed
